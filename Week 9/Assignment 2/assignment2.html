Practice Set, Week 9, Event Handling and Bubbling

This page contains two buttons, each with an event handler that causes the button, when clicked to echo its text to the page.

Your task is to rewrite the event handling code so that there is only one event handling function. That function should be called when a user clicks on either button. The output should remain identical.

You'll want to consider attaching a single event handler to a node further up the DOM tree, and checking there for which element was actually clicked (using a conditional).
